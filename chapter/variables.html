Chapter 2: Variables

Variables in C

Often it is useful to be able to store data for use in your program. Data can be stored in variables. There are several different types of variable, which can each be used to store different types of data.
These are the main types of variable.

C SyntaxFull NameSize (bytes)Description
intinteger4Holds an integer value
charcharacter1Holds a chracter
floatfloating point4Holds a floating point number (number with decimals)
doubledouble precision floating point8Can have more after the decimal place than a floating point

The sizes of each variable are machine dependent (ie different computers may use different amounts of memory). To find the size of a particular variable, use the sizeof function (see below). These sizes are important as they define size limits on values stored in variables, but more on this later.


Let's use some variables:
  1. #include <stdio.h>
  2.  
  3. int main(int argc, char *argv[]) {
  4.     int number = 4;
  5.     char letter;
  6.  
  7.     printf("The value of number is: %d\n", number);
  8.  
  9.     number = number + 2; //add 2 to the number
  10.     printf("The value of number is: %d\n", number);
  11.  
  12.     number += 2; //+= means "increment by".
  13.                  //this line has the same effect as line 8
  14.                  //the "+" can be replaced by other signs
  15.     printf("The value of number is: %d\n", number);
  16.  
  17.     number *= 3; //multiply by 3
  18.     printf("The value of number is: %d\n", number);
  19.  
  20.     number++; //++ means "increment by 1"
  21.               //-- can be used to decrement
  22.     printf("The value of number is: %d\n", number);
  23.  
  24.     number = number % 2; // % is the syntax for modulo
  25.     printf("The value of number is: %d\n", number);
  26.  
  27.     letter = 'a';
  28.  
  29.     printf("\nletter: %c\n", letter);
  30.     printf("The ASCII value of letter is: %d\n", letter);
  31.  
  32.     return 0;
  33. }

The output from that is:
The value of number is: 4
The value of number is: 6
The value of number is: 8
The value of number is: 24
The value of number is: 25
The value of number is: 1
 
letter: a
The ASCII value of letter is: 97

On lines 4 and 5 we declare the variables. The syntax for declaring a variable is "type name;". Optionally, the variable can be assigned a value at this point by using "type name = value;". After a variable is declared, it can be assigned a value "name = value;". The new value replaces its old value.

In all the printf lines, you will notice %d and %c. These are called "placeholders". printf can do far more than just print static text to standard output. It can be used to print static values, variable values, or return values of functions. Each type of variable has its own placeholder. The first argument to printf must be a string. This string can contain placeholders. The following arguments correspond, in order, to the the placeholders in the string. There can be as many placeholders in the string, as long as exactly the same number of values are specified as the following arguments. %d is a placeholder for integers. %c is for characters. On lines 29 and 30, you can see how using a different placeholder can print a different value for the same variable.

Return Values

Firstly, let's loosely define "expression" as something which can be evaluated with a value.
When a function is said to return a value, the evaluation of the function will result in this value.
The expression "4+4" evaluates to 8, so the code:
  1. int n;
  2. n = 4+4;

will result with n taking the value 8.
Similarly, a function call is an expression, as long as it has a return value (as some function do not), it can be treated as any other expression. If there was a function "halve()" which returns half of it's argument, the following code:
  1. int n;
  2. n = halve(8);

would result in n taking the value 4. The same operators can be used on functions as on any other number or variable. In order of operations, functions are evaluated before operators. In the case of nested functions:
  1. int n;
  2. n = halve(halve(8)); //this will put the value 2 in n

the inner-most function is evaluated first.

The sizeof function

The sizeof function can be used to calculate the number of bytes used to store a particular variable. This function take a variable of any type as an argument, and returns an unsigned long int (this will be explained later). The placeholder for an unsigned long int is %lu.
  1. #include <stdio.h>
  2.  
  3. int main(int argc, char *argv[]) {
  4.     int number;
  5.     char letter;
  6.  
  7.     printf("Size of integer: %lu\n"
  8.             "Size of character: %lu\n"
  9.             "Size of float: %lu\n"
  10.             "Size of double: %lu\n", 
  11.             sizeof(number), sizeof(letter),
  12.             sizeof(float), sizeof(double));
  13.  
  14.     return 0;
  15. }

The output:
Size of integer: 4
Size of character: 1
Size of float: 4
Size of double: 8

In that code you can see how the return value functions can be passed as arguments to othur functions. Also, it shows how you can use white space to spread you code out and make it more readable.

Symbolic Constants

Avoid using "magic" numbers. In the examples so far, the code has contained static numbers. For example, on line 4 in the example at the top of the page: "int number = 4;", it is unclear why the number 4 is used. When making a program use constants rather than placing numbers in the code. This will make code easier to read. People reading your code will know what the numbers mean. Also, if the number needs to be changed, you need only change the definition of the constant once, rather than every instance of the number throughout your code.

To define a constant, place the line "#define NAME value" at the top of the file, just below the #include lines. Constant names must be in all upper case. Here is an example of a program with constants. %f is the placeholder for floating points.
  1. #include <stdio.h>
  2.  
  3. #define PI 3.14159
  4. #define RADIUS 2
  5.  
  6. int main(int argc, char *argv[]) {
  7.  
  8.     float area = PI * RADIUS * RADIUS;
  9.  
  10.     printf("area: %f\n", area);
  11.  
  12.     return 0;
  13. }

Output:
area: 12.566360

When the code is compiled, the references to symbolic constants are exactly replaced with their value. Constants are purely for readibility, but they are still important.
