Chapter 5: Loops

Introduction to Loops

With what we have covered so far, the "flow" of our programs is one way only. Each line is evaluated at most once. However it is very useful to be able to do certain actions multiple times. This is achieved with loops. The two structures of loops we will look at are while loops and for loops.

While Loops

These are syntatically identical to if statements, except the keyword "if" is replaced with "while". The condition is checked, the code inside is evaluated if true, and repeat. The loop will only exit when the condition is false. Here is an example of some code with a while loop.
  1. #include <stdio.h>
  2.  
  3. #define ANSWER 42
  4. #define MAXTRIES 10
  5. #define FALSE 0
  6. #define TRUE 1
  7.  
  8. int main(int argc, char *argv[]) {
  9.     int guess;
  10.     int solved = FALSE;
  11.     int tries = MAXTRIES;
  12.  
  13.     printf("Try to guess the number (%d tries left): ", tries);
  14.  
  15.     while (!solved && tries > 0) {
  16.         scanf("%d", &guess);
  17.         tries--;
  18.  
  19.         if (guess > ANSWER) {
  20.             printf("Too high. ");
  21.         } else if (guess < ANSWER) {
  22.             printf("Too low. ");
  23.         } else {
  24.             printf("Correct!\n");
  25.             solved = TRUE;
  26.         }
  27.  
  28.         if (solved) {
  29.             printf("You got it in %d tries.\n", MAXTRIES - tries);
  30.         } else if (tries > 0) {
  31.             printf("Guess again (%d tries left): ", tries);
  32.         } else {
  33.             printf("No tries left.\nThe answer was: %d\n", ANSWER);
  34.         }
  35.  
  36.     }
  37.  
  38.  
  39.     return 0;
  40. }
  41.  

output
Try to guess the number (10 tries left): 4
Too low. Guess again (9 tries left): 50
Too high. Guess again (8 tries left): 40
Too low. Guess again (7 tries left): 42
Correct!
You got it in 4 tries.

Notice the variable "solved". This is an example of a flag. A flag is a variable that is usually either set to 0 (false) or 1 (true) and often used to help evaluate a logical expression in a condition. In this case, the flag is initially false, as the user has not yet successfully guessed the number. The loop will terminate when the flag is set to true, or the number of tries left reaches 0. When the user guesses correctly, the flag is set to true.

Flags can also be used in if statements if you want to check how a logical expression was evaluated later in the code. Rather than re-evaluating the expression, you can set a flag in the if statement, then just check the value of the flag later on.

For Loops

For loops are a special type of loop for a particular (but common) situation. This is when a counter is required and something needs to be done a certain number of times. You will probably find that you use these far more than while loops. Everything that can be done with a for loop can be done with a while loop, but for loops provide an elegant, readible approach to this particular case. Here is an example:
  1. #include <stdio.h>
  2.  
  3. #define SIZE 12
  4.  
  5. int main(int argc, char *argv[]) {
  6.     int table;
  7.     int value;
  8.     int i;
  9.  
  10.     printf("Which times table would you like: ");
  11.     scanf("%d", &table);
  12.  
  13.     for (i = 1;i <= SIZE;i++) {
  14.         value = i*table;
  15.         printf("%d x %d = %d\n", table, i, value);
  16.     }
  17.  
  18.     return 0;
  19. }

output
Which times table would you like: 4
4 x 1 = 4
4 x 2 = 8
4 x 3 = 12
4 x 4 = 16
4 x 5 = 20
4 x 6 = 24
4 x 7 = 28
4 x 8 = 32
4 x 9 = 36
4 x 10 = 40
4 x 11 = 44
4 x 12 = 48

The part in the brackets in the first line of a for loop has 3 parts seperated by semicolons.
The first part is what happens before the for loop starts iterating.
The second part is a condition that while true, the for loop will iterate.
The third part is something that happens after each iteration.

The common case is to use these parts to maintain a counter variable that ensures that the for loop iterates a certain number of times.
In the first part you initialise the variable. (i=0)
In the second part you give a condition that become false after a certain number of iterations.(i<10)
In the third part you make the counter "count". (i++)

The counter variable must be declared before the for loop. "i, j, k..." are common naming conventions for these types of counter variables. The counter variable can be accessed from inside the for loop. After the loop exits, the value of the counter will be such that it doesn't satisfy the condition.

Nesting Loops

It may seem trivial, but you can nest as many structures inside other structures as you like. You can use this to make you code extremely powerful. Keep in mind that nesting too many loops can make for a very slow program, as the number of iterations is multiplied as you nest.

Infinite Loops

Always be careful to make sure that your loops will definitely terminate at some point. If a loop in your program does not terminate, press Ctrl+c and start debugging.
