Chapter 10: Structures

Structures

Ever wanted to create your own type of variable? Structs allow you to group variables together to form what are essentially new types of variables. The struct keyword is used to create new structs. Here is an example of code that uses structs.
  1. #include <stdio.h>
  2. #include <string.h>
  3.  
  4. #define MAXLEN 30
  5.  
  6. struct date {
  7.     int day;
  8.     int month;
  9.     int year;
  10. }; //don't forget this semicolin
  11.  
  12. struct person {
  13.     struct date birthday;
  14.     char firstname[MAXLEN];
  15.     char surname[MAXLEN];
  16.     char gender;
  17. }; //everyone always forgets this semicolin
  18.  
  19. int main(int argc, char *argv[]) {
  20.  
  21.     struct person student;
  22.  
  23.     student.gender = 'm';
  24.  
  25.     strcpy(student.firstname, "Steve");
  26.     strcpy(student.surname, "Roberts");
  27.  
  28.     student.birthday.day = 12;
  29.     student.birthday.month = 4;
  30.     student.birthday.year = 1992;
  31.  
  32.     return 0;
  33. }

That example doesn't output anything, but shows the syntax to use when dealing with structs. They are useful to group together variables that store data about concepts or things in the real world. Oh, and don't forget the semicolin after the close parenthesis. It will break all your code. If you find one day that all your code is broken, this should be one of the first places you look.

typedef

Having to type the word 'struct' every time we want to use our home made variables is kind of a nuisance. Fortunately, there is a workaround. The keyword typedef is used to create 'synonyms' for data types. It is mainly used to make one word synonyms for struct variables. The convention for synonyms is to start with a capital letter, then camel case. Here is the example above, with typedefs.
  1. #include <stdio.h>
  2. #include <string.h>
  3.  
  4. #define MAXLEN 30
  5.  
  6. typedef struct date Date;
  7.  
  8. struct date {
  9.     int day;
  10.     int month;
  11.     int year;
  12. }; //don't forget this semicolin
  13.  
  14. typedef struct person Person;
  15.  
  16. struct person {
  17.     Date birthday;
  18.     char firstname[MAXLEN];
  19.     char surname[MAXLEN];
  20.     char gender;
  21. }; //everyone always forgets this semicolin
  22.  
  23. int main(int argc, char *argv[]) {
  24.  
  25.     Person student;
  26.  
  27.     student.gender = 'm';
  28.  
  29.     strcpy(student.firstname, "Steve");
  30.     strcpy(student.surname, "Roberts");
  31.  
  32.     student.birthday.day = 12;
  33.     student.birthday.month = 4;
  34.     student.birthday.year = 1992;
  35.  
  36.     return 0;
  37. }
